**LACUNEX AI** is a professional, high-performance, and secure AI ecosystem designed for seamless interaction with the world's most advanced language and vision models. It provides a premium user experience with end-to-end encryption, deep reasoning capabilities, and integrated image workflows.

---

## ✨ Features

- **🔐 End-to-End Encryption**: Total privacy for your conversations. Messages are encrypted/decrypted only in your browser—the server never sees your raw data.
- **🧠 Deep Reasoning Mode**: Toggle "Reasoning" to engage logical thinking models, perfect for complex problem-solving and coding.
- **🎨 Imagine Engine**: Generate high-quality visuals directly in the chat using `/imagine [prompt]`.
- **🖼️ Visual Analysis**: Upload images for instant analysis, transcription, and contextual understanding.
- **📁 Smart Workspace**: Organize multiple conversations with a sleek, searchable sidebar and workspace-level stats.
- **🚀 Model Library**: Dynamic access to top-tier models (Llama 3.3, Gemini 2.0, Groq, etc.) with real-time provider switching.
- **📄 Professional Export**: Seamlessly export your critical data to **PDF**, **Word (DOCX)**, or **Excel (XLSX)**.

## 🚀 Getting Started

1. **Clone the repo:**
   ```bash
   git clone https://github.com/shasradha/LACUNEX-AI.git
   ```
2. **Install dependencies:**
   * Frontend: `cd frontend && npm install`
   * Backend: `cd backend && pip install -r requirements.txt`
3. **Setup environment variables:**
   * Create `.env` files in both `frontend/` and `backend/` with your API keys.
4. **Run the platform:**
   * Backend: `python -m uvicorn main:app --reload`
   * Frontend: `npm run dev`
