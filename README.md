## ⚙️ The Core Architecture

### 1. Multi-Provider Key Rotation Engine 
Standard AI platforms fail when they hit server capacity. **LACUNEX never sleeps.**
Powered by a fault-tolerant, auto-rotating distribution engine, LACUNEX intelligently routes queries across **5 Elite Providers** utilizing **28 independent API keys**:

| Provider | Purpose | Speed/Specs |
|----------|---------|-------------|
| **Cerebras** | Instant Token Generation | Wafer-Scale Engine (WSE) ~ 1000+ T/s |
| **Groq** | Sub-second algorithmic reasoning | LPU-driven architecture |
| **Gemini** | Mass data analysis | Gemini 2.5 Flash / Pro level token limits |
| **SambaNova** | Deep reasoning execution | 405B Parameter Tier |
| **OpenRouter**| Seamless Fallback | Aggregates 200+ models |

*If one key hits a rate limit? The engine silently pivots to the next in milliseconds. The user experiences actual zero downtime.*

### 2. MAX OUTPUT Engine (Document Mode)
Typical AI systems hit token ceilings after ~1000 words. LACUNEX obliterates this ceiling through its proprietary **Max Output Pipeline**.
- Detects intent for large-scale documentation.
- Autonomously drafts a 20-chapter Table of Contents.
- Executes a multi-pass asynchronous loop.
- Generates **150+ page textbooks** natively in the browser with structured Markdown, embedded Mermaid diagrams, and zero hallucination drops.

### 3. Linguistic Omniscience & Intent Matrix
Why force humanity to speak perfect English to machines?
- The **v5.0 NLP Pre-Processor** analyzes raw text to detect intent behind misspellings, abbreviations, and mixed cultural dialects.
- **Hinglish, Banglish, Tenglish** are parsed flawlessly. If a user asks *"bhai yeh quantum physics samjhao"*, LACUNEX mirrors the tone, language, and cultural context natively.

## 💻 The Ultimate UI/UX Workspace

Designed iteratively until it felt like a **slick $1,000/month Enterprise SaaS**, LACUNEX features deep Glassmorphism, 24px background blurs, multi-stop gradients, and micro-animations on every interaction element.

### Terminal Zero (Code Studio)
An entire IDE built right into the platform.
- **Multi-Language Execution**: Run Python, Java, C++, Go, Rust, and 50+ other languages directly in the browser via secure backend tunneling.
- **Live Sandboxes**: Build web apps entirely in chat. The Artifact Viewer seamlessly bundles HTML, CSS, and JS and renders them live via a secured iframe. Zero deployment latency.
- **Sync-Scroll Highlights**: An ultra-premium code viewer mimicking VS Code using synchronized overlay mechanics for buttery-smooth repository exploration.

### Lacunex Flow 
A revolutionary, node-based **visual canvas** that breaks free from linear chat logic.
Drag, drop, and wire together specialized AI nodes to form complex data pipelines—routing output from one AI directly into the context window of another. *(Built exclusively for expansive Desktop ecosystems).*

## 📦 Tech Stack & Performance Details

**A 30-Day Sprint by ONE Developer:**
In exactly one month, Shasradha mapped, built, and launched the full stack:

- **Backend (API)**: FastAPI / Python serving concurrent asyncio requests, SSE (Server-Sent Events) streaming, and complex Rate limiters.
- **Frontend (UI)**: Next.js 14 utilizing extreme Vanilla CSS overriding, Framer Motion for organic fluid movement, and intelligent State persistence via `localStorage`.
- **Infrastructure**: Advanced micro-services segmentation (`intent_detector.py`, `ai_router.py`, `search_service.py`), allowing independent API scaling.
